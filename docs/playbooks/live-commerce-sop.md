# AI Live Commerce SOP

## Before live

- Pick 10-20 products with complete proof status.
- Prepare market-specific live rundown.
- Prepare FAQ for condition, delivery, payment, and proof.
- Set rules for holds, VIP price, and payment deadline.

## During live

- Live Host Agent presents condition, sizing, flaws, and styling.
- Live Moderator Agent captures SKU requests and buyer questions.
- Risky questions are escalated to the founder.

## After live

- Cut clips for TikTok/Reels/Shorts.
- CRM Agent follows up buyers by SKU.
- Analytics Agent records watch time, comments, DMs, and sales.
