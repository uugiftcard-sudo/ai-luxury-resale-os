# Daily SOP

## Morning

- CEO Agent creates the daily battle plan.
- Market Manager UK/HK checks platform priorities.
- Buyer Agent reviews sourcing leads.
- Analytics Agent reports best products, best content, and risks.

## Afternoon

- Proof Agent completes missing proof packs.
- Listing Agent publishes or refreshes listings.
- Content Agent creates short-video hooks and captions.
- Community Agent handles real engagement and VIP drop warm-up.

## Evening

- Live Host Agent prepares the live rundown.
- Live Moderator Agent handles questions and sends risky issues to the founder.
- CRM Agent follows up warm leads, old buyers, and VIP buyers.

## Night

- Fulfilment proof is saved: item details, packing video, payment proof, tracking.
- Finance Agent reports cashflow and ROI.
- Dispute Agent reviews any risky messages before sleep.
